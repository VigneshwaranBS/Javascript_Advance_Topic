## Run Locally

Clone the project

```bash
  git clone https://github.com/VigneshwaranBS/Javascript_Advance_Topic
```

Go to the project directory

```bash
  cd Javascript_Advance_Topic
```

Install dependencies

```bash
  npm install
```

